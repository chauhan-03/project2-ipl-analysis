# 🏏 IPL Data Analysis Dashboard

An interactive data analysis dashboard exploring **15+ years of IPL match data** with visual insights on teams, venues, toss decisions, and season trends.

## 📊 Features

- 🏆 **Team Win Analysis** — Top teams by total wins across seasons
- 🎯 **Toss Decision Insights** — Bat vs Field preference breakdown
- 📅 **Season Trends** — Match count and patterns over the years
- 🌆 **Venue Analysis** — Most active cities/stadiums
- 🔧 **Interactive Filters** — Filter by season range
- 📋 **Raw Data Explorer** — View underlying data

## ⚙️ Setup & Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/chauhan-03/ipl-data-analysis
cd ipl-data-analysis

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
streamlit run app.py
```
